# Talking to Machines: Chrome Extension Workshop Pack

This folder contains 13 ready-to-use prompts. Each one, pasted into Claude.ai free tier, will produce a working Chrome extension as a downloadable zip.

## How students use these

1. Pick a prompt file from this folder
2. Open claude.ai in a browser (free tier is fine)
3. Copy the entire contents of the prompt file
4. Paste into a new Claude conversation and send
5. Claude will create the extension files and package them as a downloadable zip
6. Download, unzip, then follow the "Load unpacked" steps included in Claude's response

## Workshop flow suggestion

Open with Tab Rescue as a shared first build so the whole room succeeds together. Then let students pick from the rest based on their own domain.

## Troubleshooting

If an extension does not work on first install, have the student paste the exact error message back to Claude with "This did not work. Please fix." Claude will usually correct it within one or two iterations.

If a student hits the free tier message cap, keep Google AI Studio open as a fallback and reshare the same prompt there.
